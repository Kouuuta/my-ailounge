# Intern Logs

This directory contains workspaces maintained by interns and contributors.

The purpose of this directory is to keep personal documentation, research, task tracking, and notes organized in a single location.

## Suggested Structure

```text
intern-logs/
├── contributor-name/
│   ├── README.md
│   ├── TODO.md
│   ├── RESEARCH.md
│   ├── CHANGELOG.md
│   └── NOTES.md
```

## Contents

Each contributor workspace may contain:

* TODO lists
* Research notes
* Progress logs
* Documentation drafts
* Personal findings related to AI Lounge

This structure is currently a proposal and may evolve as the project grows.
